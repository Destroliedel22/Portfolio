You are capture by the robots and need to escape. You find a meltgun and a shield. You use this to fight the robots that will come to kill you. You can also find a big sword and a hammer you can use. The robots sometimes drop gears that you can use on the doors so you can progress further to escape. You got the last gear and now you can finally escape and party.

About Project

Game Engine: Unity

Code language: C#

Platforms: Quest 2 and 3

Time: 4 weeks

People: 2 devs/5 artists

Description

This was the first time i worked together with game artists and we had to make a shooter game. The artist already had a great idea to make flying robots shooting you. so we wroked around that. Then we had the idea to make door a that open when you kill the last enemy of the wave, but you do need to see that you need to put the gear you got from the last enemy in the door. Then we made this game and it was a lot of fun. Some artists are really good and can deliver very quick. That was my first EXPERIENCE(:>) with Game artists.
