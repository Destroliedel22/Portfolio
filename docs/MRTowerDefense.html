Defhander


You need to defend your kingdom cause you are the god of your kingdom. The monsters try attacking it but you defend it with your turrets and hand. The monsters keep coming and coming until you finally defeated them all and you live in peace.

About Project

Game Engine: Unity

Code language: C#

Platforms: Quest 3

Time: 4 weeks

People: 3 devs

My Experience

We had to make a game with different hardware that we ae used to. So we made a game with the Oculus Quest 3 in MR. That is when you can see the real world and the game world at the same time. We wanted to make a tower defense so we made a platform where everything happens on. You can place turrets and also have abilities to kill the monsters on the board. I made this with 2 other developers and now we know how to set up a game for MR for the quest 3. I learned a lot about MR and it was fun.

My part was setting up MR, the hand animations, being able to move the whole playfield around and lock the playfield.
