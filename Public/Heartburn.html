You are a scientist and you found out you were frozen inside a crypod and finally woke up. You realize that you need to keep moving otherwise you will slowly die. So you start exploring and found out you are in an abandoned factory where there are still some weird flying robots roaming around. You start killing them but soon find out your in a infinite loop of killing these things and there is no way out.

About Project

Game Engine: Unity

Code language: C#

Platforms: Quest 3

Time: 8 weeks

People: 3 dev 2 art

My Experience

This project was all about publishing our game. The process of making this game was hard because we got a lot of bugs we couldn't fix for a couple days sometimes weeks. It became a really fun game at then end though and i really enjoyed the fun when it was finally done. I made this game together with 3 devs and 2 artists although there is not much art the game is really fun to play.

I made the forcebubble shader that you can see in the 2nd picture, hand animations, the roguelike system, the weapons and the staminabar.
